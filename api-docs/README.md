## POC Grape API - Hello World

This is a example to hello world with Grape.

### Getting starting

#### Dependencies

- Ruby 3+
- Rails 7

#### Install dependences

```
bundle install
```

#### Start application

```
rails server
```

#### Testing

In browser: http://localhost:3000/api/hello?name=Catharina

Return:

```json
{"msg":"Hello Catharina"}
```
