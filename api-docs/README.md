## POC Grape API - Docs

This is a example to documentation with Grape and Swagger.

### Getting starting

#### Dependencies

- Ruby 3+
- Rails 7

#### Install dependences

```
bundle install
```

#### Start application

```
rails server
```

#### Testing

In browser: http://localhost:3000/api-docs
